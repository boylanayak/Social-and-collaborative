# SOCIAL AND COLLABORATION
Форум для студентов вуза: Темы, теги, модерация; бэк — поиск, уведомления, ролей (студент/препод).

## How to Install NPM and Run NPM (Step by Step)
1. Install Node.js (NPM is included)
2. Open Git Bash
   npm -v
   npm install
   npm run dev
